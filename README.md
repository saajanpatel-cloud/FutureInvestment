# FutureInvestment

Thematic equity **research** workflows (multi-sector latent demand, picks-and-shovels mapping, optional Finnhub market context, adversarial bear-case gate). See [`.claude/CLAUDE.md`](.claude/CLAUDE.md) and [`.claude/skills/thematic-equity-research/SKILL.md`](.claude/skills/thematic-equity-research/SKILL.md). Outputs go under `research/memoes/` and `research/watchlists/` (tracked placeholders included). Open **`research/watchlists/SINGLE_SCREEN_REPORT.html`** in a browser for a one-page combined run (summary, pillars, **316-ticker** manifest + rubric from `universe_manifest.csv`, map, Finnhub context slot, adversarial, watchlist). **Refresh data and HTML:** `./scripts/refresh_watchlists.sh`. To append the curated universe-expansion sleeve (already applied once): `python3 scripts/fi_double_universe.py` then refresh again. Python helpers live in [`scripts/`](scripts/) (`fi_snapshot.py`, `fi_finnhub_context.py` when `FINNHUB_API_KEY` is in `.env`, `fi_mentions.py`, `fi_embed_single_screen.py`, …). Copy [`.env.example`](.env.example) → `.env` and register a free [Finnhub](https://finnhub.io/register) key for analyst/insider/news context on the shortlist. Load [`watchlist-ui/index.html`](watchlist-ui/index.html) for a local watchlist editor.

Originally scaffolded via [`scripts/new-claude-project.sh`](../../scripts/new-claude-project.sh). Repo-root hooks apply when working from `AI_Projects`.
